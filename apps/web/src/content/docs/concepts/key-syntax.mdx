---
title: Key syntax & bindings
description: How to write keyboard shortcut bindings in Tactile — combos, sequences, the mod alias, and named keys.
---

A binding is a string. Tactile parses it into one or more chords.

- **Combos** — join keys with `+`: `mod+shift+k`, `ctrl+alt+delete`
- **Sequences** — separate strokes with spaces: `g i`, `ctrl+k ctrl+s`
- **`mod`** — resolves to `⌘` on macOS and `Ctrl` everywhere else, so you write one binding for both
- **Alternatives** — pass an array: `['mod+k', 'ctrl+p']`

```ts
kb.add({ id: 'save', keys: 'mod+s', handler: save });
kb.add({ id: 'inbox', keys: 'g i', handler: goToInbox });      // press g, then i
kb.add({ id: 'open', keys: ['mod+o', 'mod+p'], handler: open }); // either works
```

## Modifier aliases

| Alias | Modifier |
| --- | --- |
| `ctrl`, `control` | Control |
| `shift` | Shift |
| `alt`, `option`, `opt` | Alt / Option |
| `cmd`, `command`, `meta`, `win`, `super` | Meta / Command |
| `mod` | Meta on macOS, Control elsewhere |

## Named keys

`enter`, `esc`, `space`, `tab`, `backspace`, `delete`, `up`/`arrowup`, `down`, `left`, `right`,
`home`, `end`, `pageup`, `pagedown`, `f1`–`f24`, and more. Symbols can be written directly (`?`, `/`,
`.`) or spelled out (`plus`, `slash`, `comma`).
