import { Tabs, TabItem } from '@astrojs/starlight/components';

Install the core engine, and the React adapter if you're using React.

<Tabs syncKey="pkg">
  <TabItem label="npm">
    ```bash
    # framework-agnostic core
    npm install @tactile-js/core

    # React hooks (pulls in core)
    npm install @tactile-js/react
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @tactile-js/core

    pnpm add @tactile-js/react
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @tactile-js/core

    yarn add @tactile-js/react
    ```
  </TabItem>
</Tabs>

`@tactile-js/react` lists `react >= 17` as a peer dependency.

Both packages ship ESM + CJS builds with TypeScript declarations, are tree-shakable
(`sideEffects: false`), and have zero runtime dependencies beyond the core.
